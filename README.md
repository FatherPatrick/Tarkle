# Tarkle - Tarkov Wordle

A Wordle-style word guessing game themed around Escape from Tarkov.

## How to Play

- Guess the hidden **5-letter Tarkov word** in **6 attempts**.
- After each guess, the tiles change colour to show how close your guess was:
  - 🟩 **Green** – correct letter in the correct position
  - 🟨 **Yellow** – correct letter in the wrong position
  - ⬛ **Grey** – letter is not in the word

You can type using your physical keyboard or tap the on-screen keyboard.

## Getting Started

```bash
npm install
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
npm run build
npm run preview
```
